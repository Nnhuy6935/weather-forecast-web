# weather_web

- to build web
  ```sh
  flutter build web
  ```
- to run web
  ```sh
  flutter run
  ```
## Technical 
- ``Shared Preferences`` : save data and show again when user first access to website
- ``Dio`` : manage HTTP request  
- ``go_router`` : web navigation

# Deploy web 
- to deloy web
  
  Step 1: Build web
  ```sh
  flutter build web
  ```
  Step 2: Deploy
  ```sh
  firebase deploy
  ```
